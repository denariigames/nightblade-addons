# NPC Subtitle

<img src="https://github.com/denariigames/nightblade-addons/blob/master/NPCSubtitle/dist/screenshot.png" alt="NPCSubtitle" height="350">

An addon for NightBlade MMO that adds a subtitle field to NPC entities. Useful if you want to show a role below the name.

## Usage

Edit your existing CanvasNpcUI prefab and replace the UI Npc Entity component

-or-

Reference CanvasNpcSubtitleUI prefab in GameInstance
